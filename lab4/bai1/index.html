Cú pháp CSS

Một quy tắc CSS bao gồm một bộ chọn và một khối khai báo.

h1 { color: blue; font-size: 12px; }

Bộ chọn trỏ tới phần tử HTML mà bạn muốn định dạng.

Khối khai báo chứa một hoặc nhiều khai báo được phân tách bằng dấu chấm phẩy.

Mỗi khai báo bao gồm tên thuộc tính CSS và giá trị, được phân tách bằng dấu hai chấm.

Nhiều khai báo CSS được phân tách bằng dấu chấm phẩy và các khối khai báo được bao quanh bởi dấu ngoặc nhọn.

Ví dụ

Trong ví dụ này, tất cả các phần tử <p> sẽ được căn giữa và có màu chữ đỏ:

p {
  color: red;
  text-align: center;
}

Ví dụ giải thích

  • p là một bộ chọn trong CSS (nó trỏ đến phần tử HTML mà bạn muốn định dạng: <p>).
  • color là một thuộc tính, và red là giá trị thuộc tính.
  • text-align là một thuộc tính, và center là giá trị thuộc tính.
Bạn sẽ tìm hiểu thêm về bộ chọn CSS và thuộc tính CSS trong các chương tiếp theo!
